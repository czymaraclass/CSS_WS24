# Computational Social Science

## Course Description

This research training introduces a range of computational data analysis methods. Using R and occasionally Python, students learn to write scripts for analyzing text-as-data and other data types. The course covers both supervised and unsupervised machine learning techniques for classification, topic modeling, and sentiment analysis.

This research training was originally taught at Goethe University Frankfurt but is now openly accessible to anyone. Most sessions combine lectures with hands-on tutorials.
