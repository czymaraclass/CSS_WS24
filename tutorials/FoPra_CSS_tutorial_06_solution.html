b) Preprocessing

To prepare the text data for analysis:

  1. Convert text to lowercase
  2. Tokenize the text
  3. Remove punctuation, numbers, symbols, and common stopwords
  4. Retain only words that appear in a reasonable number of documents

The following code achieves this:

movie_ratings <- corpus(tolower(movie_ratings_raw$review),
                        docvars = movie_ratings_raw)

toks_movie_ratings <- tokens(movie_ratings,
                           remove_punct = TRUE,
                           remove_numbers = TRUE,
                           remove_symbols = TRUE,
                           remove_separators = TRUE,
                           include_docvars = T)

toks_movie_ratings <- tokens_remove(toks_movie_ratings,
                                  stopwords())

toks_movie_ratings <- tokens_trim(toks_movie_ratings,
                                min_docfreq = 0.01,
                                max_docfreq = 0.5,
                                docfreq_type = "prop")

movie_ratings_dfm <- dfm(toks_movie_ratings)

2. Data Splitting

We will split the dataset into training and test sets. The training set (80% of the data) will be used to train the model, and the test set (20%) will evaluate its performance.

a) Create Indices

Generate a random set of indices for the training data. This set of indices should correspond to 80 percent of the full DFM (e.g., row numbers that shall correspond to the training data).

set.seed(1337) # Reproducibility

train_indices <- sample(1:ndoc(movie_ratings_dfm), size = 0.8 * ndoc(movie_ratings_dfm))

b) Split the Data

Use the indices to create separate training and test DFMs (training: indices numbers; test: not indices numbers). Also store the observed values of sentiment for the training and the test data by using docvars(dfm_train, "sentiment") and docvars(dfm_test, "sentiment"), respectively.

# Training data
dfm_train <- movie_ratings_dfm[train_indices, ]

observed_train <- docvars(dfm_train, "sentiment")

# Test data
dfm_test <- movie_ratings_dfm[-train_indices, ]

observed_test <- docvars(dfm_test, "sentiment")

4. Unsupervised Machine Learning: Wordfish

Now we want to turn to a Wordfish model to predict the similarity of reviews, based on a one-dimensional latent variable. For more information on this method, see Slapin and Proksch (2008).

a) Subset Data

Wordfish models can be quite slow with large, high-dimensional data. To speed things up, use the dfm_sample() function to draw a random sample of 2,000 texts from the movie_ratings_dfm.

movie_ratings_dfm_sample <- dfm_sample(movie_ratings_dfm, size = 2000)

b) Run Wordfish Model

Use quanteda.textmodelsâ textmodel_wordfish() function to run a Wordfish model based on this sample. The values estimated for the latent variable is stored in the theta variable of the Wordfish object. Have a look at the first couple of values.

wordfish_model <- textmodel_wordfish(movie_ratings_dfm_sample)

head(wordfish_model$theta)
## [1] -0.29508466  0.08126594 -0.52255494  1.18627512  0.51616601  1.43581179

c) Interpret Features (\(\beta\))

\(\beta\) (stored in wordfish_model$beta) is the association of each word to the latent scale. Put differently, words with highly positive or negative \(\beta\) values strongly influence the documentâs position on the latent variable. We want to see how a few selected positive and negative words are correlated with the latent variable. To this end, load the quanteda.textplots package and adapt the following code:

textplot_scale1d(wordfish_model, margin = "features",
                 highlighted = c("... (positive words)",
                                 "... (negative words)"))

Do the positive and the negative words cluster at different ends of the scale?

d) Interpret Documents (\(\theta\))

\(\theta\) (stored in wordfish_model$theta) represents the position of each document on the latent scale, based on the words it contains. It is a continuous variable with positive and negative values (zero being the neutral point of the latent dimension). To compare the results to the observed sentiment values, create a boxplot() showing the distribution of wordfish_model$theta by the two sentiment groups. Does the plot signal a meaningful clustering along the dimension of the latent variable?

boxplot(wordfish_model$theta ~ movie_ratings_dfm_sample@docvars$sentiment,
        main = "Theta by Sentiment",
        xlab = "Sentiment",
        ylab = "Theta")
