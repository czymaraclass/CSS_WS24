1. Scraping Wikipedia

We want to scrape data from Wikipedia. In particular, we want to scrape information for Wikipediaâs page of Frankfurt. Check out the content of the page. We will first extract some information from the Wikipedia page and then scrape metadata from the API.

a) Scraping Wikipedia Pages

To to scrape the Wikipedia page of Frankfurt, we need to access the page and extract the relevant information. Use the read_html() function of the rvest package to read the HTML content of the page.

wiki_url <- "https://en.wikipedia.org/wiki/Frankfurt"

wiki_page <- read_html(wiki_url)

b) Extracting Information

Now we can extract information from the Wikipedia page using CSS selectors. For example, we can extract all paragraphs of the page. Use the SelectorGadget to identify the CSS selector for the paragraphs. Then use the html_nodes() function to select the elements and html_text() to extract the content.

frankfurt_wiki_nodes <- html_nodes(wiki_page, "p")

frankfurt_wiki_text <- html_text(frankfurt_wiki_nodes)

2. Scraping Wikipedia Metadata

We can also scrape metadata from the MediaWiki API. For example, we can scrape the daily page views of the Wikipedia page of Frankfurt between 01 Jan 2016 and today.

a) Access the API

We use the GET() function from the httr package to access the API. The URL for the API request is https://wikimedia.org/api/rest_v1/metrics/pageviews/per-article/en.wikipedia/all-access/all-agents/Frankfurt/daily/2016010100/2024121200.

wiki_url_02 <- GET("https://wikimedia.org/api/rest_v1/metrics/pageviews/per-article/en.wikipedia/all-access/all-agents/Frankfurt/daily/2016010100/2024121200")

b) Extract Data

We can extract the data from the API response using the content() function. Use the content() function to parse the data. Then use bind_rows() to combine the parsed data into a data frame.

wiki_meta_data <- content(wiki_url_02, "parsed")

wiki_meta_data <- bind_rows(wiki_meta_data$items)

c) Prepare and Plot the Data

Convert the timestamp variable to a date format using lubridate (ymd(substr(wiki_meta_data$timestamp, 1, nchar(wiki_meta_data$timestamp)-2))) and plot the daily page views.

wiki_meta_data$date <- ymd(substr(wiki_meta_data$timestamp, 1, nchar(wiki_meta_data$timestamp)-2))

# plot
wiki_meta_data %>%
  ggplot(aes(x = date,
             y = views)) +
  geom_line()

3. Scraping the IC2S2 Website

Next, we want to scrape information from the website of the 9th International Conference on Computational Social Science. We will scrape the session titles and chairs.

a) Inspect the Website

  • Open the ic2s2 website in a browser
  • Use SelectorGadget to inspect the elements you want to scrape
  • Identify the HTML tag and classes or IDs of the elements for:
    • The session titles
    • The session chairs
# SelectorGadget: (only) session titles are .col-12-medium > h2
# SelectorGadget: (only) session chairs are h2 i

b) Scrape the Website

Use read_html() to fetch the IC2S2 program website.

ic2s2_url <- "https://ic2s2-2023.org/program"

ic2s2_page <- read_html(ic2s2_url)

c) Parse and Extract Session Data

Use html_nodes() to select the elements identified in 3.a); then use html_text() to extract the text. Do this once for the session titles and once for the names of the session chairs.

ic2s2_session_titles <- ic2s2_page %>%
  html_nodes(".col-12-medium > h2") %>%
  html_text()

ic2s2_session_chairs <- ic2s2_page %>%
  html_nodes("h2 i") %>%
  html_text()

d) Combine Session Data

Combine the title and chair name data into one data frame. Who chaired more than one session at the conference?

ic2s2_sessions <- data.frame(
  titles = ic2s2_session_titles,
  chairs = ic2s2_session_chairs)
